# Sales & Profit Analysis Dashboard (Tableau)

## Project Overview
This project is a Tableau dashboard built to analyze sales performance and profit trends across different regions, categories, and customer segments. It helps identify key business insights for better decision-making.

## Objective
To understand:
- Which regions are performing well in sales and profit
- Which product categories are most and least profitable
- How discounts impact profit
- Overall sales trends over time

## Dataset Used
Sample Superstore dataset (CSV/XLSX format)

## Tools Used
- Tableau Desktop
- Microsoft Excel (for dataset handling)

## Key Insights
- Some regions show high sales but low profit due to heavy discounts
- Technology category performs better compared to Furniture and Office Supplies
- Certain sub-categories consistently generate losses
- Profit varies significantly based on discount levels
